# ⚛️ Quantum Superposition Using Hadamard Gate

## 📌 Project Overview

This project demonstrates the concept of quantum superposition using a single qubit and the Hadamard gate. The quantum circuit is created and simulated using the Qiskit framework. By applying the Hadamard gate to a qubit initialized in the |0⟩ state, the system enters a superposition where the qubit has equal probability of being measured as 0 or 1.

The project helps illustrate fundamental concepts in quantum computing such as qubits, quantum gates, superposition, and measurement.

---

## 🚀 Features

* Simulation of a single qubit quantum circuit
* Application of the Hadamard gate
* Visualization of quantum measurement outcomes
* Demonstration of quantum superposition

## ⚙️ Installation

1. Clone the repository

```
git clone https://github.com/your-username/quantum-superposition-project.git
```

2. Navigate to the project folder

```
cd quantum-superposition-project
```

3. Install required libraries

```
pip install qiskit
```
